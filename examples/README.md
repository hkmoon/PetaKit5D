# Example Notebooks

This directory contains example Jupyter notebooks demonstrating PetaKit5D usage.

## Notebooks

1. **01_basic_io.ipynb** - File I/O operations (TIFF, Zarr)
2. **02_image_filtering.ipynb** - Image filtering examples
3. **03_deconvolution_workflow.ipynb** - Complete deconvolution pipeline
4. **04_stitching_example.ipynb** - Tile stitching workflow

## Running the Notebooks

Install Jupyter:

```bash
pip install jupyter
```

Start Jupyter:

```bash
jupyter notebook
```

Navigate to this directory and open a notebook.

## Requirements

All examples require PetaKit5D to be installed:

```bash
pip install petakit5d[all]
```

Some examples may require additional data files. These will be generated in the notebook or can be downloaded from the repository's data folder.
