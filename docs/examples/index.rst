Overview
--------

The examples demonstrate common workflows and best practices for using PetaKit5D:

1. **Basic I/O**: Reading and writing microscopy data
2. **Image Filtering**: Applying various filters to images
3. **Deconvolution**: Complete deconvolution workflow
4. **Stitching**: Tile-based volume reconstruction

All example notebooks are available in the ``examples/`` directory of the repository.
